### 🤖 AI-Augmented Development

I embed AI across the SDLC, with a human review gate at every stage:

| Stage | How I use AI |
|-------|-------------|
| **Requirements & design** | Draft and pressure-test specs, edge cases, and acceptance criteria before coding |
| **Code generation** | Claude Code, Copilot, and Cursor for scaffolding, refactors, and boilerplate |
| **PR & code review** | AI-assisted review to surface bugs, security issues, and standards drift early |
| **Testing** | Generate unit and integration tests to lift coverage, especially on legacy code |
| **Ops & incidents** | AI-assisted log analysis and postmortem summaries |

I maintain custom instruction files, prompt standards, and responsible-AI guardrails
(hallucination checks, human-in-the-loop approval) so AI accelerates delivery without
compromising quality.
